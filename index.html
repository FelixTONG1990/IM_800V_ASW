如何解决401错误？

  1. 访问 GitHub Token设置页面
  2. 点击 "Generate new token" 按钮
  3. 为令牌命名（例如 "文件下载器"）
  4. 选择过期时间（建议选择 "No expiration"）
  5. 勾选 "repo" 权限（这是必须的）
  6. 点击底部的 "Generate token" 按钮
  7. 复制生成的令牌并粘贴到上面的输入框中
  8. 点击 "应用令牌" 按钮
